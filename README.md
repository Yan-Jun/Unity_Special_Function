# ClickChangeColor
依照顏色順序，點擊平滑修改顏色，並記錄起來用於下次開啟初始顏色。

![](https://github.com/Yan-Jun/Unity_Special_Function/blob/master/Gifs/ClickChangeColor.gif) 


# LerpMoveToAllPosition
加入要移動的座標，依照座標順序，在設定的時間內跑完所有座標。
